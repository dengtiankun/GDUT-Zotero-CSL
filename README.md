# GDUT-Zotero-CSL

广东工业大学本科毕设 Zotero 参考文献样式（修复版）。

根据开源项目 [zotero-chinese](https://github.com/zotero-chinese/styles) 修改，修复了以下问题：

1. 按照语言显示“等”或“et al.”；
2. 姓名恢复姓氏全大写以符合广工规范；
3. 仅纯电子资源显示引用日期和 URL；
4. 无 DOI；
5. 修正期刊分隔符为全角逗号；
6. 修复 et al. 与全角句号重叠的问题。

> **注：** 本样式按照《2026届毕业设计（论文）手册》规范修改，不保证未来的时效性和准确性，请同学们以当届最新要求为准。

## 🚀 安装与使用教程

### 方法一：一键在线安装（最简单）
1. 确保你的电脑上正在运行 Zotero软件。
2. 点击https://github.com/dengtiankun/GDUT-Zotero-CSL/raw/refs/heads/main/gdut-thesis.csl
3. Zotero 界面会自动弹出提示框，询问是否安装该样式，点击 **确定** 即可。

### 方法二：手动本地安装
1. 下载本项目中的 `.csl` 格式文件到你的电脑本地。
2. 打开 Zotero，依次点击顶部菜单栏：**编辑 (Edit)** -> **首选项 (Preferences)** -> **引用 (Cite)** -> **样式 (Styles)**。
3. 点击样式列表下方的 **`+`** 号，选中你刚刚下载的 `.csl` 文件，导入即可使用。

## 💡 其他文献管理软件使用教程

本项目提供的 `.csl` (Citation Style Language) 格式文件是一种通用的开源引文标准。除了 Zotero，它还完美兼容 **Mendeley**、**Paperpile** 等主流文献管理软件。

### 📌 Mendeley 导入教程
1. 打开 Mendeley Desktop 或 Mendeley Reference Manager。
2. 在顶部菜单栏依次选择 **View** -> **Citation Style** -> **More Styles...**。
3. 切换到 **Get More Styles** 标签页。
4. 将本项目的 **Raw 链接** 粘贴到 "Download style" 输入框中，点击 **Download**。
   *(注：部分版本也支持直接将本地的 `.csl` 文件拖拽进去)*
5. 回到 **Installed** 标签页，搜索“广东工业大学”，点击 **Use this Style** 即可应用。

### 📌 Paperpile 导入教程
1. 点击浏览器右上角的 Paperpile 插件图标，点击齿轮进入 **Settings** (设置)。
2. 在左侧导航栏中选择 **Citation Styles**。
3. 点击 **Upload CSL file**，选择你下载到本地的 `.csl` 文件进行上传。
4. 在使用 Google Docs 或 Word 插件写作时，搜索并选中刚上传的广工样式即可。

### 📌 其他软件 (通用方法)
如果你的文献管理软件支持 CSL 标准格式，通常只需要进入软件的 **“引文样式 (Citation Style / Formatting)”** 设置，寻找 **“添加自定义样式 (Add Custom Style)”** 或 **“从文件导入 (Import from file)”** 选项，选中下载到本地的 `.csl` 文件即可完成安装。
